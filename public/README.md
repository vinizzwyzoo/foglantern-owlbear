# Fog & Lantern Controller – Owlbear Rodeo

Extensão para:
- controlar opacidade do Fog of War
- criar lanternas (luzes) em tokens

## Como usar
1. Hospede o projeto (Vite, Netlify, GitHub Pages)
2. Copie a URL do `manifest.json`
3. Owlbear Rodeo → Profile → Extensions → Add Extension
4. Cole a URL e ative na sala

## Requisitos
- Ser GM da sala
